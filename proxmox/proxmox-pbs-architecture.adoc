= NetApp ONTAP 사용한 Proxmox 백업 서버 아키텍처에 대해 알아보세요.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Proxmox Backup Server(PBS)는 NetApp ONTAP 스토리지와 통합되어 Proxmox Virtual Environment(VE) 워크로드에 엔터프라이즈급 백업 기능을 제공합니다. 이 아키텍처는 ONTAP SnapMirror 복제를 사용하여 증분 백업, 중복 제거, 압축, 암호화 및 재해 복구를 제공합니다.

Proxmox Backup Server는 Proxmox 가상 환경에 맞게 설계된 엔터프라이즈급 백업 기능을 제공합니다. PBS는 증분 백업, 중복 제거, 압축 및 암호화를 포함한 기능을 통해 VM 및 컨테이너에 대한 효율적이고 안정적인 백업 및 복원 작업을 제공합니다.

PBS는 로컬 스토리지, NFS, iSCSI 및 S3 호환 객체 스토리지를 포함한 다양한 스토리지 백엔드를 지원합니다. PBS는 NetApp ONTAP 스토리지와 통합될 경우 고성능의 확장 가능한 스토리지를 활용하여 백업 데이터 저장소 및 오프사이트 백업 대상으로 사용할 수 있습니다.



== 솔루션 아키텍처

이 솔루션은 다음과 같은 구성 요소로 이루어져 있습니다.

* *Proxmox VE 클러스터* - 가상화 기능을 제공하고 여러 노드에 걸쳐 가상 머신(VM) 및 컨테이너를 관리합니다.
* *Proxmox 백업 서버* - ONTAP 스토리지와의 통합을 통해 Proxmox VE 워크로드에 특화된 백업 및 복원 기능을 제공합니다.
* * NetApp ONTAP 스토리지 * - Proxmox VE 클러스터 및 PBS 백업 데이터 저장소에 고성능의 확장 가능한 스토리지를 제공합니다.
* *네트워크 인프라* - Proxmox VE 노드, PBS 및 ONTAP 스토리지 간의 저지연, 고처리량 연결을 보장합니다.
* * NetApp Console* - 여러 NetApp 스토리지 시스템 및 데이터 서비스를 중앙에서 관리할 수 있도록 지원합니다.
