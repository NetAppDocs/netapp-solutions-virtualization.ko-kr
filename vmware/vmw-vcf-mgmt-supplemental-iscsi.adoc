= VCF 관리 도메인에 iSCSI 데이터 저장소를 보조 저장소로 추가하기 위한 배포 워크플로
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
VMware Cloud Foundation(VCF) 관리 도메인의 보조 스토리지로 iSCSI 데이터스토어를 추가하는 작업을 시작해 보세요.  iSCSI를 위한 논리적 인터페이스(LIF)를 갖춘 스토리지 가상 머신(SVM)을 설정하고, ESXi 호스트에서 iSCSI 네트워킹을 구성하고, ONTAP tools for VMware vSphere 배포하고, VMFS 데이터 저장소를 생성합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"]link:vmw-vcf-mgmt-supplemental-iscsi-requirements.html["배포 요구 사항 검토"]
[role="quick-margin-para"]
VCF 관리 도메인에 보조 스토리지로 iSCSI 데이터 저장소를 추가하는 데 필요한 요구 사항을 검토합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["둘"]link:vmw-vcf-mgmt-supplemental-iscsi-svm-lifs.html["SVM 및 LIF 생성"]
[role="quick-margin-para"]
iSCSI 트래픽을 위해 여러 개의 LIF가 있는 SVM을 생성합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["삼"]link:vmw-vcf-mgmt-supplemental-iscsi-network.html["네트워킹 구성"]
[role="quick-margin-para"]
ESXi 호스트에서 iSCSI에 대한 네트워킹을 설정합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["네"]link:vmw-vcf-mgmt-supplemental-iscsi-storage.html["저장소 구성"]
[role="quick-margin-para"]
ONTAP 도구를 배포하고 사용하여 스토리지를 구성합니다.
